Tools: Python, Pandas, NumPy, Matplotlib, Power BI
Environment: Jupyter Notebook

Project Overview

This project analyzes Airbnb listing data to understand pricing trends, booking behavior, and revenue opportunities across neighbourhoods.
Using Python for data cleaning and analysis and Power BI for visualization, the project transforms raw datasets into an interactive dashboard that helps identify high-performing locations and listing types.

Problem Statement

Airbnb hosts and investors often struggle to determine:

which neighbourhoods are most profitable

optimal pricing strategies

which listing types receive the most bookings

Without proper data analysis, pricing and investment decisions may rely on assumptions rather than data.

This project analyzes Airbnb data to generate data-driven insights for better decision making.

Project Objectives

Analyze pricing trends across neighbourhoods

Understand distribution of listing types

Evaluate booking behavior and demand

Identify high revenue locations

Compare Superhost vs Normal host performance

Key Insights

Entire homes dominate the Airbnb marketplace.

Premium neighbourhoods show significantly higher prices.

A small number of neighbourhoods generate the majority of revenue.

Higher priced listings tend to have lower booking rates.
